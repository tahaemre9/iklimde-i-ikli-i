# Yapay Zeka Destekli Web Site
### Amaç:
iklim değişikliğinin yıkıcı etkilerini insanlara göstermek ve bunları önlemek için neler yapacaklarını öğretmek
## Ne yapacağım?
öncelikle iklim değişikliğiyle ilgili bilgiler içeren web sitesi yapacağım bu web sitesinde iklim değişikliğinin nedenlerini,yıkıcı etkilerini
gösteren ve kullanıcının merak ettiği diğer soruları yanıtlayan bir yapay zeka olacak
## Yapay zeka ne işe yarayacak?
yapay zekaya görsel atıldığında görseldeki sorunun ne olduğunu, nasıl ve neden olduğunu en son nasıl önlenebileceğini gösterecek
## Web sitesinde neler olacak?
Web sitesinde iklim hakkında bilgiler
